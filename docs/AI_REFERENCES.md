# AI Reference Sources

Use these primary references when changing service integrations or asking AI tools
for implementation help. Prefer official documentation over copied examples.

## qBittorrent

- qBittorrent Wiki: https://github.com/qbittorrent/qBittorrent/wiki/
  - Use for WebUI setup, authentication, reverse proxy behavior, and troubleshooting.
- qBittorrent WebUI API: https://github.com/qbittorrent/qBittorrent/wiki/WebUI-API
  - Use for endpoint paths, request methods, parameters, and response behavior.

## Sonarr

- Sonarr API Docs: https://sonarr.tv/docs/api/#v3/description/introduction
  - Use for Sonarr v3 endpoint paths, request methods, parameters, schemas, and responses.
- Servarr Sonarr API Wiki: https://wiki.servarr.com/sonarr/api
  - Use for API authentication notes, especially `X-Api-Key` behavior.
